변수 : 값을 담는 공간

변수는 어떤 값을 담을 수 있는 상자라고 생각하면 된다.

변수의 종류
const : 변수의 값을 변경할 수 없음 (진공포장까지 완료한 박스)
let : 변수의 값을 변경할 수 있음 (아직 포장하지 않은 박스)

변수 선언 방법


    

    const 변수이름;
    const 변수이름 = 값;
    let 변수이름;
    let 변수이름 = 값;

    사용 예시
    const naTreeum = 'Handsome';
    let age = 10;

변수이름 설정 시 주의사항
: 숫자로 시작할 수 없다
: 띄어쓰기 할 수 없다

고기분쇄기로 고기를 갈고 있다고 생각하자.
먼저 고기분쇄기에 고기를 넣으면 고기가 갈린다.
소고기는 갈린 소고기로, 돼지고기는 갈린 돼지고기로, 닭고기는 갈린 닭고기로 나온다.


이 구조에서 "고기분쇄기"는 "함수 이름"에 해당한다.
소고기, 돼지고기, 닭고기는 모두 "입력값"에 해당한다.
갈린 소고기, 갈린 돼지고기, 갈린 닭고기는 모두 "반환값"에 해당한다. 

function 고기분쇄기(접시)  { 
//고기분쇄기라는 함수를 만들면, 접시에 올려진 고기를 갈기 시작한다 
    고기를 간다(접시);  //고기가 갈린다
    return 갈린고기; 갈린 고기를 반환한다
}

고기분쇄기(소고기); //접시에 소고기를 올려서 고기분쇄기를 작동시킨다. 아래의 나머지도 각각 돼지고기, 닭고기를 넣고 작동시킨다.
고기분쇄기(돼지고기);
고기분쇄기(닭고기);
